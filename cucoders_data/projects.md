## 🧾 ABK OPE (Client-Server Accounting System)
### 📸 Project Preview
![ABK Ope](https://res.cloudinary.com/dmujvqbqx/image/upload/v1776193321/abk-ope-auth_idjteh.jpg)
![ABK Ope - Short Video] (https://res.cloudinary.com/dmujvqbqx/video/upload/v1776193430/abk-ope_rt1vig.mp4)

### 🧠 Description
A client-server desktop accounting system developed in Python for financial management operations. The system was designed to handle accounting workflows and data management for an institutional environment (Embassy of Niger).

It includes a separated architecture with a server-side backend responsible for data processing and storage, and a client-side application used by operators to interact with the system.

### 🏛️ Context
Developed as a real-world institutional software solution for accounting operations in a diplomatic environment.

### 🛠️ Tech Stack
Python · Socket Programming / API Communication · SQL (MariaDB / SQLite) · Desktop GUI (Tkinter / PyQt)

### ⚙️ Features
- Client-server architecture (distributed system)
- Secure data communication between client and server
- Financial records management and tracking
- Structured accounting workflow system
- Role-based operational usage (server/client separation)
